## Tags
### Standart Tag
```PHP
<?php
//code
?>
```
### Echo Tag (Since 5.4)
```PHP
<?=
//code
?>
```
### Short Tag (deprecated)
```PHP
<?
//short_open_tags = On in php.ini to activate it
//code
?>
```
### Script Tag (deprecated)
```PHP
<script language="php">
//code
</script>
```
### ASP Tag (deprecated)
```PHP
<%
//code
%>
<%=
//code
%>
```
